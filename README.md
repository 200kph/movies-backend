# Movies test task server

## Description

NestJS server for Movies Web app

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Deploy
```bash
# test environment
npm run deploy:test

# prod environment
npm run deploy:prod

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
